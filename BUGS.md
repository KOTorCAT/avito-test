# Баг-репорты

## БАГ-001: API /api/1/item всегда возвращает 400 "поле likes обязательно"

| Поле | Значение |
|------|----------|
| **Серьёзность** | Critical |
| **Приоритет** | Highest |
| **Компонент** | API / Создание объявления |
| **Окружение** | https://qa-internship.avito.com |

### Краткое описание
При попытке создать объявление через POST /api/1/item сервер всегда возвращает ошибку "поле likes обязательно", даже когда поле `likes` присутствует в запросе в любом формате.

### Шаги воспроизведения
1. Отправить POST запрос на `/api/1/item` с телом:
```json
{
  "sellerId": 123456,
  "name": "Test",
  "price": 100,
  "likes": 0,
  "viewCount": 0,
  "contacts": 0
}
